h5table

The jQuery replacement for select boxes

h5table gives you a customizable select box with support for searching, tagging, remote data sets, infinite scrolling, and many other highly used options.

In your language

h5table comes with support for RTL environments, searching with diacritics and over 40 languages built-in.

Remote data support

Using AJAX you can efficiently search large lists of items.

Fits in with your theme

Fully skinnable, CSS built with Sass and an optional theme for Bootstrap 3.

Fully extensible

The plugin system allows you to easily customize h5table to work exactly how you want it to.

Dynamic item creation

Allow users to type in a new option and add it on the fly.

Full browser support

Support for both modern and legacy browsers is built-in, even including Internet Explorer 8.

Getting started with h5table

In order to use h5table, you must include the JavaScript and CSS file on your website. You can get these files built for you from many different locations.

Using h5table from a CDN

h5table is hosted on both the cdnjs and jsDelivr CDNs, allowing you to quickly include h5table on your website.

  1. Include the following lines of code in the <head> section of your HTML. 

    <link href="//cdnjs.cloudflare.com/ajax/libs/h5table/4.0.1/css/h5table.min.css" rel="stylesheet" />
<script src="//cdnjs.cloudflare.com/ajax/libs/h5table/4.0.1/js/h5table.min.js"></script>
    Immediately following a new release, it takes some time for CDNs to catch up and get the new versions live on the CDN.

  2. Initialize h5table on the <select> element that you want to make awesome. 

    <script type="text/javascript">
  $('select').h5table();
</script>
  3. Check out the examples page to start using the additional features of h5table.

Downloading the code locally

In some situations, you can't use h5table from a CDN and you must include the files through your own static file servers.

  1. Download the code from GitHub and copy the dist directory to your project.

  2. Include the following lines of code in the <head> section of your HTML. 

    <link href="path/to/h5table.min.css" rel="stylesheet" />
<script src="path/to/h5table.min.js"></script>
  3. Check out the examples page to start using the additional features of h5table.

The different h5table builds

h5table provides multiple builds that are tailored to different environments where it is going to be used. If you think you need to use h5table in a nonstandard environment, like when you are using AMD, you should read over the list below.

Build name When you should use it
Standard (h5table.js / h5table.min.js) This is the build that most people should be using for h5table. It includes the most commonly used features.
Full (h5table.full.js / h5table.full.min.js) You should only use this build if you need the additional features from h5table, like the compatibility modules or recommended includes like jquery.mousewheel
