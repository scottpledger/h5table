Localization, RTL and diacritics support

Multiple languages

h5table supports displaying the messages in different languages, as well as providing your own custom messages that can be displayed.

The language does not have to be defined when h5table is being initialized, but instead can be defined in the [lang] attribute of any parent elements as [lang="es"]. 





  
RTL support


  

    h5table will work on RTL websites if the dir attribute is
    set on the <select> or any parents of it. You can also
    initialize h5table with dir: "rtl" set.
