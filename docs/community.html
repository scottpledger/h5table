Community

h5table has an active community that enjoys helping each other out.

Getting support

Having trouble getting h5table working on your website? Is it not working together with another plugin, even though you think it should? h5table has a few communities that you can go to for help getting it all working together.

  1. Try sending a message to the h5table discussion group or searching the archives.
  2. Ask in the #h5table channel on chat.freenode.net or use the web irc client.
  3. Raise a question on Stack Overflow, make sure to include the "jquery-h5table" tag.

Reporting bugs

Found a problem with h5table? Feel free to open a ticket on the h5table repository on GitHub, but you should keep a few things in mind:

  1. Use the GitHub issue search to check if your issue has already been reported.
  2. Try to isolate your problem as much as possible, so we can easily test if the issue has been fixed.
  3. Once you are sure the issue is with h5table, and not a third party library, submit a ticket to the repository.

You can find more information on reporting bugs in the contributing guide, including tips on what information to include.

Requesting new features

New feature requests are usually requested by the h5table community on GitHub, and are often fulfilled by fellow contributors.

  1. Use the GitHub issue search to check if your feature has already been requested.
  2. Check if it hasn't already been implemented as a third party plugin.
  3. Please make sure you are only requesting a single feature, and not a collection of smaller features.

You can find more information on requesting new features in the contributing guide.

Getting involved

You can find more information on getting involved with h5table in the contributing guide.
